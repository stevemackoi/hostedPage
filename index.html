Process Details

process arguments

/proc/PID/cmdline

process current working directory (symlink)

/proc/PID/cwd

path to actual process executable file (symlink)

/proc/PID/exe

environment used by process

/proc/PID/environ

the root path as seen by the process

/proc/PID/root

basic information about a process including run state and memory usage

/proc/PID/status

hard links to any tasks that have been started by this (the parent) process

/proc/PID/task

socket table (in hex)

/proc/PID/net/tcp

systemd Service Control

List of currently running services

systemctl list-units

Start a service

systemctl start <service>

Stop a service

systemctl stop <service>

Restart a service

systemctl restart <service>

Restart only if already running

systemctl condrestart <service>

Reload a service

systemctl reload <service>

Status of a service

systemctl status <service>

Enable service at boot

systemctl enable <service>

Disable service at boot

systemctl disable <service>

Check if service is enabled at boot

systemctl is-enabled <service>

Check to see if service is running

systemctl is-active <service>

Kill all processes related to a service

systemctl kill <service>

View service configuration file

systemctl cat <service>
